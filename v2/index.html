installation

Achordion is 10 HP wide Eurorack module. It is powered by -12V/+12V 2x5 connector. The red stripe (-12V) should be connected on the side of the board marked with the white line. The boxed power connector should make it easier to connect with the right polarity.

controls, inputs and outputs

WAVETABLE (A) knob selects the wavetable. When the button (Y) is held down, this knob scrolls through wavetable banks.

TONE (B) knob selects root note that should be playing. When the button (Y) is held down, this knob is used to select tonic of the scale.

DETUNE (C) knob controlls spread of secondary voices and amount of detune. When the button (Y) is held down, this knob controlls the scale mode.

CHORD (D) knob selects the chord or interval that should be playing. When the button (Y) is held down, this knob selects the mode in which chords are built.

On the left side, there is a column of LED diodes (X) that are visualizing the currently selected parameter.

Below the LEDs, there is a button (Y) that allows access to alternative parameters of each of the knobs.

CV inputs TONE (1) and TONE' (2) used to control the root note that is playing on the main and side line, respectively. When TONE is connected, the TONE knob controlls octave offset. These inputs follow 1V/oct standard, between 0 and +10 V.

CV inputs TABLE (3), DETUNE (4) and CHORD (5) are used to control the same attributes as the knobs described above. The value selected by the knob is added to the value set through CV. These inputs span between -5 and +5 V.

Audio outputs OUT (6) and OUT' (7) play the main and the side line respectively. When only one is connected, the outputs are mixed together.

wavetables

There are 37 wavetables to select from, divided into 4 banks to select from:

  1. Perfect – classic waveforms: triangle, sine, square and saw. All of these are containing pure harmonics.
  2. Harsh – full and distorted sounds. These can get very full when played in rich chords.
  3. Soft – cleaner sounds and bells.
  4. Sines – the bottom half of this bank is a sequence of sin function multiplications providing variety of different sounds from clean to crazy noises. The upper half contains sums of sin functions, sliding through this part provides a low-pass filter effect.

The transitions between wavetables within a single bank are gradual and very smooth, suitable for live control through ADSR.

The bank can be selected by pressing the button (Y) while turning the WAVETABLE (A) knob.

detune

This parameter controls number of secondary voices per each played note and their detune. The knob/CV input scrolls through three of these detune modes:

  1. Disabled, only primary tones of the selected chord are playing.
  2. Single sub-octave, the first degree of the chord is duplicated one octave lower. This is a classic sub-octave.
  3. Each tone in the chord is duplicated once with a unisono. This produces slight movement or vibrato effect, depending on the amount of the detune.
  4. Two sub-octaves per each degree. This can produce very rich wall of sound and gets crazy with lot of detune.

While moving through each of the modes, secondary voices are being gradually put them of tune. In lower detunes, the difference in frequency is very small and produces "beats" - making the sound moving and warm, when it is pushed higher it turns into a wild cacophony.

calibration

While inputs TONE and TONE' should match the 1V/oct standard, they may be little off, either due to used components or the CV source not matching the standard preciselly. It is possible to calibrate each of these inputs individually for their CV source. To do so, follow this procedure:

  1. While pressing the button, connect jack to the input.
  2. The left column of LEDs should lit up.
  3. Play the note C on the CV source and press the button.
  4. Now the right column of LEDs should lit up.
  5. Play the note C one octave higher and press the button again.

reset

To reset calibration and all saved secondary parameters (wavetable bank, scale tonic, scale mode, chord mode), press the button while powering on the module.
