installation

Achordion is 10 HP wide Eurorack module. It is powered by +12V/-12V 2x5 connector. The red stripe (-12V) should be connected on the side of the board marked with the white line. The boxed power connector should make it easier to connect with the right polarity.

controls, inputs and outputs

On the left side, there is a column of LED diodes (X) that are visualizing the currently selected parameter. Below the LEDs, there is a button (Y) that allows access to alternative parameters of each of the knobs.

WAVETABLE (A) knob selects the wavetable. When the button (Y) is held down, this knob scrolls through wavetable banks.

TONE (B) knob selects root note that should be playing. When the button (Y) is held down, this knob is used to select tonic of the scale.

DETUNE (C) knob controlls spread of secondary voices and amount of detune. When the button (Y) is held down, this knob controlls the scale mode.

CHORD (D) knob selects the chord or interval that should be playing. When the button (Y) is held down, this knob selects the mode in which chords are built.

CV inputs TONE (1) and TONE' (2) used to control the root note that is playing on the main and side line, respectively. When TONE is connected, the TONE knob controlls octave offset. These inputs follow 1V/oct standard, between 0 and +10 V.

CV inputs TABLE (3), DETUNE (4) and CHORD (5) are used to control the same attributes as the knobs described above. The value selected by the knob is added to the value set through CV. These inputs span between -5 and +5 V.

Audio outputs OUT (6) and OUT' (7) play the main and the side line respectively. When only one is connected, the outputs are mixed together.

two voices

The module has two voices, primary and secondary. The pitch of these voices can be controlled independencly while all the other attributes are shared.

The primary voice is controlled through the TONE knob and CV input. The primary voice can play a chord, controlled through the CHORD knob and CV input.

The secondary voice is controlled through the TONE' CV input and is playing only when a jack is connected. Unlike the primary voice, the secondary is not playing chords.

wavetables

There are 37 wavetables to select from, divided into 4 banks to select from:

  1. Perfect – classic waveforms: triangle, sine, square and saw. All of these are containing pure harmonics.
  2. Harsh – full and distorted sounds. These can get very thick when played in rich chords.
  3. Soft – cleaner sounds and bells.
  4. Sines – the bottom half of this bank is a sequence of sine function multiplications providing variety of different sounds from clean to crazy noises. The upper half contains sums of sine functions, sliding through this them provides a low-pass filter effect.

The transitions between wavetables within a single bank are gradual and very smooth, suitable for live control through ADSR.

The bank can be selected by pressing the button (Y) while turning the WAVETABLE (A) knob.

detune

This parameter is set through the DETUNE knob and CV input and controls number of secondary voices per each played note and their detune. The knob/CV input scrolls through three of these detune modes:

  1. Disabled, only primary tones of the selected chord are playing.
  2. Single sub-octave, the first degree of the chord is duplicated one octave lower. This is a classic sub-octave.
  3. Each tone in the chord is duplicated once with a unisono. This produces slight movement or vibrato effect, depending on the amount of the detune.
  4. Two sub-octaves per each degree. This can produce very rich wall of sound and gets crazy with lot of detune.

While moving through each of the modes, secondary voices are being gradually put them of tune. In lower detunes, the difference in frequency is very small and produces "beats" - making the sound moving and warm, when it is pushed higher it turns into a wild cacophony.

quantization

Quantizing makes sure that anything that comes out of the module stays in tune. Thanks to this, you can spend less time thinking about the harmony, easily play with other musicians, and more.

To select root note of the scale, hold the button while turning the TONE knob. LEDs I through VII then symbolize notes of C major: C, D, ..., B. The # LED signalizes that the note is sharp.

To select the scale mode, hold the button while turning the DETUNE knob. LEDs I through VII then symbolize each of the modes. Selected mode affects the mood or character of the tune you are playing:

  1. Ionian (major)
  2. Dorian
  3. Phrygian
  4. Lydian
  5. Mixolydian
  6. Aeolian (minor)
  7. Locrian

If you are not a trained musician, don't despair, keep the default root note and pick one of the modes (I personally really like Dorian), just use you ears. If you know basics of music theory, being able to select the scale may enable you to combine this module with other instruments or other musicians.

chords

This module allows playing chords of up to 5 degrees, each of them affected by the detune.

In order to visualize selected chords, LEDs are displaying the degrees of them. For example, if the selected chord is basic fifth, LEDs I, III and V light up to signalize that first, third and fifth are playing. If this does not sound like anything to you, don't worry and just use your ears.

Furthermore, there are three different modes of playing chords. These can be accessed by holding the button while turning the CHORD knob:

  1. Chords – assorted set of 19 chords. Starting with single note, followed-up with fifth, seventh, ninth and ninth inversions.
  2. Intervals – not chords per-se. In this mode, the root note is accompanied by another note of interval between minus two octaves to plus octaves.
  3. Arpeggios – four different chords that are incrementally built while the parameter is being increased. Starting with the root, adding second degree, third, ... up to five degrees.

You may wonder how to select minor or major chords. Thanks to quantization, the quality of each chord is selected automatically, based on the root note, in order to stay in tune.

split output

There are two audio outputs in the module: OUT and OUT'. When only one of these outputs is connected, it gets mixed with the unconnected one. When both outputs are connected, each plays a different line. This allows to run each line through a different effect chain.

The content of these lines depends on the playing voices. If the secondary voice is playing, OUT plays the primary voice, controlled by TONE and OUT' is playing the secondary voice, controlled by TONE'.

However, when the secondary voice is not connected, OUT contains the root note of the primary voice, while OUT' contains the rest of the chord played by the primary voice.

calibration

While inputs TONE and TONE' should match the 1V/oct standard by default, they may be little off, either due to used components or the CV source not matching the standard precisely. It is possible to calibrate each of these inputs individually for their CV source. To do so, follow this procedure:

  1. While pressing the button, connect jack to the input.
  2. The left column of LEDs should light up.
  3. Play the note C on the CV source and press the button.
  4. Now the right column of LEDs should light up.
  5. Play the note C one octave higher and press the button again.

If you don't use a keyboard as a TONE input, there is no need to calibrate.

reset

To reset calibration and all saved secondary parameters (wavetable bank, scale tonic, scale mode, chord mode), hold the button pressed while powering on the module.
